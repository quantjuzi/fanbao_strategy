# 股市术语 → pandas 代码速查表

假如你不会 Python，那么就把下面的代码背好，量化策略的代码也基本上看得懂了。

| 你说的 | pandas 写法 | 代码说明 |
|---|---|---|
| 收盘价 | df["close"] | 当天收盘价 |
| 开盘价 | df["open"] | 当天开盘价 |
| 最高价 | df["high"] | 当天最高价 |
| 最低价 | df["low"] | 当天最低价 |
| 成交量 | df["volume"] | 当天成交总股数 |
| 成交额 | df["money"] | 当天成交总金额 |
| 阳线 | df["close"] > df["open"] | 收盘价比开盘价高 |
| 阴线 | df["close"] < df["open"] | 收盘价比开盘价低 |
| 涨跌幅 | df.groupby("证券代码")["close"].pct_change() | 今天比昨天涨了百分之几 |
| 前收盘价 | df["pre_close"] | 昨天的收盘价 |
| 涨停 | df["close"] >= df["high_limit"] * 0.99 | 收盘价接近涨停价，买不进去 |
| 高开 | df["open"] > df["pre_close"] | 开盘价比昨天收盘高 |
| 低开 | df["open"] < df["pre_close"] | 开盘价比昨天收盘低 |
| 昨天最高 | df["high"].shift(1) | shift(1)=往上看一行，拿昨天的数据 |
| 前天收盘 | df["close"].shift(2) | shift(2)=往上看两行，拿前天的数据 |
| 明天开盘 | df["open"].shift(-1) | shift(-1)=往下看一行，拿明天的数据 |
| 5日均线(MA5) | df.groupby("证券代码")["close"].rolling(5).mean() | 过去5天的平均收盘价，看短期趋势 |
| 10日均线(MA10) | df.groupby("证券代码")["close"].rolling(10).mean() | 过去10天的平均收盘价 |
| 20日均线(MA20) | df.groupby("证券代码")["close"].rolling(20).mean() | 过去20天的平均收盘价，看中期趋势 |
| 上影线 | df["high"] - df[["open","close"]].max(axis=1) | 股价冲上去又下来了，顶部那根线 |
| 下影线 | df[["open","close"]].min(axis=1) - df["low"] | 股价跌下去又拉回来了，底部那根线 |
| 反包 | df["close"] > df["high"].shift(1) | 今天收盘价比昨天最高价还高 |
| 放量 | df["volume"] > df["volume"].shift(1) * 2 | 今天的成交量是昨天的两倍以上 |
| 缩量 | df["volume"] < df["volume"].shift(1) * 0.5 | 今天的成交量不到昨天的一半 |
| 创新高 | df["close"] == df["close"].rolling(20).max() | 股价突破了近20天最高点 |
| 创新低 | df["close"] == df["close"].rolling(20).min() | 股价跌破了近20天最低点 |
| 金叉 | (MA5 > MA10) & (MA5.shift(1) <= MA10.shift(1)) | 5日线上穿10日线，看涨信号 |
| 死叉 | (MA5 < MA10) & (MA5.shift(1) >= MA10.shift(1)) | 5日线下穿10日线，看跌信号 |
