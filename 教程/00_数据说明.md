
# 数据说明

## 数据来源

从聚宽（JoinQuant）平台下载的 A 股全市场日线数据。

## 数据范围

- **时间**：2026-04-12 到 2026-06-12
- **股票**：全 A 股，共 5509 支
- **格式**：CSV 文件
- **字段**：index, open, close, high, low, volume, money, pre_close, high_limit, low_limit, 证券代码, 是否涨停

## 为什么用 CSV，不用代码实时拉数据

如果用 akshare 或聚宽 API 一条条拉数据：
- 每次回测都要等几分钟下载
- 网络不稳定容易断
- 改一次参数就得重新下载一次

直接下载 CSV 的好处：
- 下载一次，重复使用
- 回测速度快，几秒出结果
- 数据稳定，不会因为网络原因出错

## 怎么从聚宽下载

1. 登录 https://www.joinquant.com
2. 进入“研究环境”（Research）
3. 新建一个 Notebook（选择 Python3 格式）
4. 去找 AI（豆包、千问、DeepSeek 等），把你要的内容发给 AI，比如"在聚宽上下载5月的全A股数据，请给我代码"
5. 运行，生成 CSV 文件
5. 然后把代码放到对话框里运行，就会生成数据 CSV 文件
6. 生成完成，到研究环境个人中心里面下载生成好的 CSV

### 聚宽下载代码

from jqdata import *
import pandas as pd

START_DATE = "2026-04-12"
END_DATE = "2026-06-12"
BATCH_SIZE = 100

stocks = get_all_securities(["stock"])
stock_list = stocks.index.tolist()

all_data = []
for i in range(0, len(stock_list), BATCH_SIZE):
    batch = stock_list[i:i+BATCH_SIZE]
    df = get_price(batch, start_date=START_DATE, end_date=END_DATE,
                   frequency="daily",
                   fields=["open","close","high","low","volume",
                           "money","pre_close","high_limit","low_limit"],
                   skip_paused=False, fq="pre", panel=False)
    if df is not None and not df.empty:
        all_data.append(df)

result = pd.concat(all_data, ignore_index=True)
result = result.rename(columns={"time":"index", "code":"证券代码"})
result["是否涨停"] = (result["close"] >= result["high_limit"] * .99).astype(int)
result.to_csv(f"全市场A股_{START_DATE}_{END_DATE}.csv", index=False, encoding="utf-8-sig")
